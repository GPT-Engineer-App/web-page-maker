- Vite
- React
- shadcn/ui

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/web-page-maker.git
cd web-page-maker
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
